Frontend Web програмирование. JavaScript

В курсе рассматриваются все необходимые теоретические и практические аспекты программирования и верстки на базе языка JavaScript, каждый и которых закрепляется лабораторной работой. Курс предполагает выполнение учебного проекта.

О курсе

  • Описание курса

    Курсы по технологиям Front End, которые проводятся в Main Academy соответствуют самым высоким международным стандартам и гарантируют высокое качество подготовки. После прохождения курса Вы будете готовы к работе в IT – компании. Курсы программирования Front End проводятся только сертифицированными инструкторами, имеющими большой практический и педагогический опыт. Обучение проводится по самым современным методикам, за счет чего обеспечивается высокая эффективность и усвояемость материала. Весь материал курса – лекции, лабораторные, практические работы, тесты и экзамены, студенты проходят в классе вместе с инструктором. Кроме этого, система управления обучением, в которой находится весь учебный материал, доступна студентам и инструкторам в режиме 24/7. С ее помощью студенты могут легко повторять материал, а инструктора управлять процессом и качеством обучения. В курсе «Программирование FrontEnd – часть 2. JavaScript» рассматриваются все необходимые теоретические и практические аспекты программирования и верстки на базе языка JavaScript, каждый и которых закрепляется лабораторной работой. Курс предполагает выполнение учебного проекта. Все материалы курса на английском языке, что дает студентам отличную возможность в освоении всей необходимой для трудоустройства и работы терминологии и подтягивает уровень владения английским.

  • Программа курса

    1. Создание логики работы со страницей 1. JavaScript базовый 1.1. Обзор языка JavaScript 1.2. Обзор инструментария средств отладки кода 1.3. Типы данных. Oператоры 1.4. Выражения. Операции над переменными. 1.5. Взаимодействие с пользователем 1.6. Использование валидатора Jshint.com 1.7. Основные конструкции программирования: ветвления, повторения 1.8. Массивы данных 1.9. Aссоциативные массивы 1.10. Стандартный класс Object, методы работы 1.11. Понятие функции 1.12. Понятие и использование замыкания и примеси 1.13. Работа со строками, датами 1.14. Конструкция try…catch 2. JavaScript продвинутый 2.1. Обзор объектной модели W3C DOM 2.2. Классы Document и Element 2.3. Обзор событий элементов в модели DOM 2.4. Объект Form 2.5. События мыши и клавиатуры для элементов управления 2.6. Кнопки 2.7. Поля ввода. Изменение регистра введенных данных 2.8. Флаги. Массивы флагов 2.9. Переключатель. Массивы переключателей 2.10. Визуальная группировка элементов 2.11. Структура URL. Использование объекта XMLHttpRequest: свойства, методы 2.12. Форматы передачи данных от сервера JSON 2.13. Cookies. Доступ к Cookies. 2.14. Технология Ajax и работа с HTTP протоколами 3. Модульный подход к разработке 3.1. Описание подхода к разработке Asynchronous Module Definition 3.2. Установка инструмента, реализующий подход RequireJS и подключение его в проект 3.3. Описание модуля с помощью define и выполнение с помощью require 3. Инструменты фронтэнд 3.1. Сборка фронтэнд проекта 3. Сертификация 4. Эффективное трудоустройство

  • Рекомендованные курсы

    Разработчик должен владеть английским на уровне не ниже Intermediate, поэтому также рекомендуем пройти тестирование на знание английского языка и, в зависимости от результатов, пройти курс “Pre-intermediate technical english (A2)”. Для эффективного освоения данного тренинга рекомендовано прохождение курса «Программирование FrontEnd – часть 1. Верстка», а также курса «Основы программирования».

  • Предварительная подготовка

    Английский язык Английский язык – желательно уровень intermediate. Чтение и понимание технических текстов, базовые навыки живого общения. Чтобы проверить свой уровень владения английским языком предлагаем пройти тест. Профессиональные качества Уверенные знания базовых web-технологий: знания HTML знание CSS базовые знания bootstrap или любого другого JS framework Чтобы проверить свои знания основ программирования на языке Java Script, предлагаем пройти тест.
